Washdb-Bot Control Panel

Backend API for controlling and monitoring washdb-bot scraper

Status: Backend running on port 5001
Database: washdb (PostgreSQL)
Note: GUI frontend not yet implemented

Available API Endpoints

General

GET /health - Health check
GET /api/info - API information

Scraper Control

GET /api/scraper/status - Get scraper status
POST /api/scraper/start - Start scraper
POST /api/scraper/stop - Stop scraper
GET /api/scraper/logs - Get logs
GET /api/scraper/config - Get configuration

Data Access

GET /api/data/companies - List companies
GET /api/data/discovery-runs - List discovery runs

Statistics

GET /api/stats/overview - Overview statistics
GET /api/stats/database - Database statistics
